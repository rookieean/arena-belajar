# Bugs pada CSS

    - margin collapsing
        margin ketumpuk dengan yang diatas, 
        jangan pakai display box, pakai display flex/grid + 

    - justify-between untuk konten dinamis
        membagi div secara horizontal, 
        kalau kontennya berubah, maka layout akan berantakan.
        gunakan flex-gap, agar konsisten gapnya.

    - menampilkan gambar
        lihat lebar panjang gambar asli, 
        object-fit: cover -> dia akan simpan rasionya, dan akan memenuhi kontainernya. object-position: 75%

    - email template
        tidak semua properti css disupport ketika membuat email template denga  HTML

        can i email, tools

    tidak ada responsive design
