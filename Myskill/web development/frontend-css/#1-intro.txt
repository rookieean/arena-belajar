CASCADE
 bagaimana browser akan memilih ada 2 aturan CSS yang sama 
 

 misal:

 button {
    color: red;
 }

 button {
    color: blue;
 }

 browser akan memilih yang blue, karena yang terbaru
 1. POA: melihat urutan kemunculan
 2. Specifity: algoritma yang memilih CSS mana yang lebih cocok
 3. Origin: urutan munculnya dari mana, browser? atau dari file
 4. Importance: beberapa CSS lebiih penting dari yang lain
