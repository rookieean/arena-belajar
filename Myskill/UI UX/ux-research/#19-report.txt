// Crafting a Questionnaire



# Questionnaire structure

    - Opening
        perkenalkan diri, jelaskan maksud survey ini dan penggunaan datanya. sebutkan juga durasi untuk menyelaraskan ekspektasi, dan sebutkan reward jika ada

    - Questions
        seluruh pertanyaan dalam reserach ini

    - Closing
        ucapan terima kasih untuk apresiasi responden

    // tipe pertanyaan: single answer & multiple answer

    Single answer
    "dari beberapa pilihan fitur berikut, fitur mana yang paling kamu sukai?"

    Multiple answer
    "Dari beberapa pilihan fitur berikut mana saja fitur yang pernah kamu gunakan?"



    // tipe pertanyaan: Rating Scale
        opsi skala dari range tertentu (1-4, 1-5, 1-10)

    skala genap
    "sangat tidak puas - tidak puas - puas - sangat puas"

    skala ganjil
    "sangat tidak setuju - tidak setuju - netral/biasa saja - setuju - sangat setuju"



    // tipe pertanyaan: matrix

    matrix
    mengetahui tingkat kepuasan seseorang terhadap beberapa tipe fitur




    // tipe pertanyaan: ranking

    Ranking 
    mengetahui urutan kepentingan dari keberadaan beberapa aspek dalam suatu fitur



    // Tipe pertanyaan: number & open-ended

    number
    mengetahui rata-rata pengeluaran user untuk membeli makanan dalam sebulan

    open-ended
    "mengapa anda rutin menggunakan fitur tersebut?"
